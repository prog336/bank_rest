# Система управления банковскими картами
Программное обеспечение, необходимое для запуска приложения:
- Docker;
- Среда разработки с терминалом.

Инструкция по запуску приложения:
1) Открыть проект, в терминале выбрать директорию "bank_rest-main".
2) Запустить Docker.
3) Прописать в терминале команду "docker compose up --build -d" или "docker compose up --build".
(Для сборки приложения может понадобиться прокси/VPN в связи с проблемами со скачиванием некоторых зависимостей из репозитория Maven в РФ)
4) Подождать загрузки зависимостей и сборки приложения и контейнеров.
5) Приложение запущено.

Swagger UI доступен по http://localhost:8080/swagger-ui/index.html
API доступен по http://localhost:8080
